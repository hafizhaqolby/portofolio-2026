## 🇮🇩 Tentang Proyek

Portofolio interaktif satu halaman yang menampilkan proyek-proyek di bidang data analytics, machine learning, dan natural language processing. Dibangun dengan HTML, CSS, dan JavaScript murni (tanpa framework), sehingga ringan dan mudah di-hosting di mana saja.

### Fitur

- **Dwibahasa (ID/EN)** — toggle bahasa Indonesia dan Inggris secara real-time tanpa reload halaman
- **Filter proyek interaktif** — proyek dikelompokkan berdasarkan kategori (ML & Data, Software & Game, Bisnis & Marketing)
- **Sertifikasi terorganisir** — 48 sertifikasi dikelompokkan ke dalam 5 kategori dalam bentuk accordion yang bisa dibuka-tutup
- **Desain responsif** — menyesuaikan tampilan di desktop maupun mobile
- **Tanpa dependency eksternal** — hanya satu file HTML, tidak perlu proses build

### Cara Menjalankan

Proyek ini tidak memerlukan proses build apa pun.

1. Clone atau download repo ini
2. Buka file `index.html` langsung di browser, **atau**
3. Deploy ke GitHub Pages: aktifkan lewat *Settings → Pages → Deploy from branch (main)*

## 🇬🇧 About This Project

A single-page interactive portfolio showcasing projects in data analytics, machine learning, and natural language processing. Built with plain HTML, CSS, and JavaScript — no framework, no build step, easy to host anywhere.

### Features

- **Bilingual (ID/EN)** — real-time language toggle, no page reload
- **Interactive project filtering** — projects grouped by category (ML & Data, Software & Game, Business & Marketing)
- **Organized certifications** — 48 certifications grouped into 5 expandable/collapsible categories
- **Responsive design** — adapts across desktop and mobile
- **Zero external dependencies** — a single HTML file, no build process required

### Running Locally

No build step required.

1. Clone or download this repository
2. Open `index.html` directly in your browser, **or**
3. Deploy via GitHub Pages: *Settings → Pages → Deploy from branch (main)*

### Tech Stack

`HTML5` `CSS3` `Vanilla JavaScript` `Google Fonts (Fraunces, Work Sans, IBM Plex Mono)`
